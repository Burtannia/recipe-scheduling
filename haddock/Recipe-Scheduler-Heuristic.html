leastFlexible :: ActionRule #

Schedule the least flexible action first. That is the action which has the smallest number of valid stations.

fastestFirst :: StationRule #

Choose the station which can perform the action the fastest.

leastIdleReq :: StationRule #

Choose the station for which the least amount of idle time would need to be added.

leastUsed :: StationRule #

Choose the station which has the fewest actions assigned to it.
