Scheduling Helper Functions

fromCapacity :: Env -> Env #

Splits a station down into one substation for each concurrent action it is capable of performing.

validStations :: IxAction -> Recipe -> Env -> [(String, Time)] #

Given an action from a given recipe and an environment, produces a list of station names paired with the time it would take each station to perform the recipe.

validStations' :: IxAction -> Recipe -> Env -> [(Station, Time)] #

Same as validStations but pairs the time with the station, not just the station name.

isValidSt :: Action -> [Action] -> Station -> Bool #

Given an action and its dependencies returns True if the given station can perform the action.

intersectStations :: [(String, Time)] -> [(String, Time)] -> [(String, Time, Time)] #

Creates the intersection of two lists of stations produced by validStations. The intersection only considers the station name hence both times are returned as they may be different.

infeasible :: Recipe -> Env -> Maybe [IxAction] #

Returns a list of actions from the given recipe that cannot be performed by any station in the given environment.

maxEnd :: Recipe -> Env -> Time #

Maximum end time of the recipe if every action was performed sequentially using the slowest station for each action.

maxDur :: Recipe -> Env -> Time #

Maximum duration of any single action in the given recipe.
